# My Notes

This is a fullstack application built on laravel 10v (restful apis) / vue.js 3 (spa)

## Features of RESTful APIs

-   Authentication with sunctum
    -   login
    -   register
    -   reset password with email (sendgrid service)
-   get all notes
-   create new note
-   update note
-   delete note
-   print note
